Description
This project was done at the end of 2017 for the Riot Games API Challenge. The challenge was to create a tool that would help introduce new players to some of the newer features in the game using Riot Games' API. Me and my partner created a two-part website. The first part was to analyze top players games and how they used these new features. We then used this data to help newer players find more optimal uses of these features. The other part was a more in-dept teaching section where we detailed what these new features do and the strategy behind when and why they are good. My group finished second in our category.

This project is a bot for a popular social program (Discord). It reads the text messages that are sent in the chat and if they are sent in a specific format, in this case preceded by a slash(/), then they would be interpreted as a command. Using this any user can feed commands to my bot and have it do various tasks. Some of these tasks include playing music, searching imgur and reddit, as well as rolling a dice amoung other things. Description
This website was created in a group of 4 for the Hack Western Hackathon. This project uses machine learning along with The Weather Network's API to predict the chances of a flight being delayed on a given day. To create this historical flight data was gathered from the past years and then filtered before being fed into a basic FeedFoward neural network. Due to short time span only outbound flights were considered with no consideration towards the weather at the destination. Also, only USA airports are selectable. Due to The Weather Network's API only being available during the hackaton this site is now retired.
